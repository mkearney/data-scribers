Pandoc and Xelatex in docker

This side project attends to use a docker image for pandoc and LaTex related stuff. TexLive is not very easy to maintain even on Ubuntu. Especially to update, since Texlive freeze every year and Ubuntu LTS versions have some retard (e.g on Ubuntu 16.04, source packages are about TexLive 2015, but TexLive servers provides 2017 packages). Plus, TexLive is really big and take a huge place in the root…
