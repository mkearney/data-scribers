tiler 0.2.0 CRAN release

The tiler package provides a map tile-generator function for creating map tile sets for use with packages such as leaflet . In addition to generating map tiles based on a common raster layer source, it also handles the non-geographic edge case, producing map tiles from arbitrary images.
